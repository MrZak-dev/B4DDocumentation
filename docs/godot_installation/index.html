Godot Installation

The Space Rider game is built using Godot 3.2.2 mono stable

Downloads

Mono sdk

  • Godot Engine mono version requires the Mono sdk to be installed on your PC so we can use The engine with .Net C# language You can download Mono Sdk From This Links

Godot Engine

  • First we have to download the game engine From this Link

Godot Engine doesn't require installation , just Download , Extract and Open . Simple !

  • After opening the exe file .

Installation

  • After Downloading Godot Engine and installing the Mono sdk now we can Import Our Game And start Editing

  • Press import button from Godot Project Manager Window

  • Go to your game folder and select project.godot file and press open

  • And finally Import and Edit

Conrats! Your Game is ready To Edit And Test
